尊厳あるデジタル社会

Humanist Tech Labは、技術決定論に異を唱えます。システムのために人間が存在するのではなく、人間のためにシステムが存在するという原点への回帰。私たちの技術は、使う人の自律性を奪うことなく、その可能性を広げるための「道具」として設計されます。

数値化できない感情、直感、そして文脈。それらを切り捨てることなく、デジタル領域において人間性を豊かに表現できるインターフェースとアルゴリズムを探求しています。

「機械は答えを出すのに役立つが、
問いを立てるのは人間の特権である」
